# Dogs vs Cats Classification using SVM

## Overview

This project implements a Support Vector Machine (SVM) model to classify images of dogs and cats. The dataset consists of images from the Kaggle Cats and Dogs dataset.

## Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-Learn
* Matplotlib

## Dataset

Kaggle Dogs vs Cats Dataset

Classes:

* Cat
* Dog

## Project Workflow

1. Load Images
2. Resize Images
3. Convert Images into Feature Vectors
4. Train-Test Split
5. Train SVM Model
6. Evaluate Model Accuracy

## Results

The model successfully classified dog and cat images using Support Vector Machine (SVM).

Accuracy Achieved:

* 51.25%
